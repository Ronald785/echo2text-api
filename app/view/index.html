Welcome to the Speech To Text API

To transcribe an audio file, send a POST request to the endpoint below with the audio file attached.

How to Use

  • Method: POST
  • Endpoint: /transcribe
  • Content-Type: multipart/form-data
  • Form field name: audio
  • Supported formats: flac, mp3, mp4, mpeg, mpga, m4a, ogg, wav, webm
  • Max file size: 100MB

Example using cURL

curl -X POST http://localhost:3000/transcribe \
     -H "Content-Type: multipart/form-data" \
     -F "audio=@path/to/audio/file.mp3"
